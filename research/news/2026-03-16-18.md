### 5. Chrome DevTools MCP 成为 HN 热门 — AI 调试集成新范式
- **来源**：[Chrome DevTools Blog](https://developer.chrome.com/blog/chrome-devtools-mcp-debug-your-browser-session)（HN 482 points, 199 comments）
- **摘要**：Google 发布了 **Chrome DevTools 的 MCP（Model Context Protocol）集成**，允许 AI Agent 直接读取和操作浏览器调试会话。这意味着 AI 编程助手可以 **直接看到运行时错误、网络请求、DOM 状态**，而不仅仅是看代码。HN 社区讨论热烈（199 条评论），主要关注这对 AI 编程工作流的颠覆性。
- **对我们的影响**：🟡 **前端开发专家应关注。** Chrome DevTools + MCP = AI Agent 可以"看到"浏览器中实际发生了什么。这对前端调试、端到端测试都是质的飞跃。我们的 Agent Browser 技能可能需要升级以利用此能力。
