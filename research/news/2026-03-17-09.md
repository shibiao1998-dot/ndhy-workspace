### 2. 🆕 VentureBeat 深度：NemoClaw 技术细节 — 单命令安装 + 本地隐私 + OpenShell 沙箱
- **来源**：[VentureBeat](https://venturebeat.com/technology/nvidias-nemoclaw-brings-privacy-and-security-controls-to-autonomous-openclaw)（3 小时前）
- **摘要**：VentureBeat 的深度报道补充了 NemoClaw 的关键技术细节：
  - **单命令安装**：`nemoclaw install` 一键部署 Nemotron 模型 + OpenShell runtime 到 OpenClaw
  - **本地优先**：可运行在 GeForce RTX PC/笔记本、RTX PRO 工作站、DGX Station、DGX Spark 上，**敏感数据无需上云**
  - **Nemotron 3 新模型族**：Ultra（5x 吞吐效率，frontier-level 智能）、Omni（音视频+语言多模态）、VoiceChat（实时语音对话）
  - **安全模型**：专门的 Nemotron safety models + 可信多模态数据检索管线
  - Jensen 原话："OpenClaw is the operating system for personal AI. This is the moment the industry has been waiting for — the beginning of a new renaissance in software."
- **对我们的影响**：🔴 **直接行动项。** (1) "本地运行 + 无需上云" 对中国市场尤其重要——数据合规 + 低延迟 + 隐私保护。(2) Nemotron 3 Ultra/Omni/VoiceChat 三件套可能成为本地部署的首选模型。(3) 我们需要评估 `nemoclaw install` 是否已可用，以及与当前 OpenClaw 版本的兼容性。

### 3. 🆕 Claude Code "2.0" 特性详解 — 多 Agent 代码审查 + /Loop 自动化 + 增强记忆
- **来源**：[Geeky Gadgets](https://www.geeky-gadgets.com/claude-code-2-code-review/)（18 小时前）
- **摘要**：社区将 Claude Code 最新特性合集称为"2.0"（非官方版本号），核心新增：
  - **"By the Way" 命令**：在不中断当前任务的情况下插入快速问题/指令，保持上下文连续性
  - **/Loop 命令**：类 cron 的定期自动执行（代码审查、依赖更新、例行维护）
  - **多 Agent 代码审查**：Team/Enterprise 计划可用，多个 AI Agent 并行审查代码、发现 bug
  - **增强记忆管理**：结构化模板存储，减少冗余，提升召回准确度
  - **Effort Level 自定义**：low/medium/high/maximum 四档，平衡深度和成本
  - **API 集成指导**：Claude Code 作为"活 API 导师"，指导 prompt caching、adaptive thinking 等高级功能
  - **交互式数据可视化**（Beta）
- **对我们的影响**：🔴 **直接影响我们的 ACP 工作流。** (1) 多 Agent 代码审查 = 我们 code-quality-gate skill 的原生替代方案。(2) /Loop 自动化 = 潜在替代部分 cron 定时任务。(3) 增强记忆管理 = 与我们的 context-management skill 方向一致，可参考其结构化模板。(4) By the Way 命令 = ACP sessions_send 的原生版本。需要确认我们的 Claude Code v2.1.37 是否已包含这些特性。
