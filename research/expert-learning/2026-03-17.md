# 专家学习建议 2026-03-17

## 🏗️ 技术架构专家
1. **MCP 上下文消耗审计**：阅读 [MCP Server Is Eating Your Context Window](https://www.apideck.com/blog/mcp-server-eating-context-window-cli-alternative)，评估我们 7 个 MCP server 的上下文开销，研究 CLI 替代方案
2. **GTC Vera Rubin 架构**：关注明早 keynote recap，特别是 Agentic AI 推理成本降 10 倍的技术原理和部署模式
3. **CPU 在 Agentic 工作流中的瓶颈**：阅读 [CNBC 报道](https://www.cnbc.com/2026/03/13/nvidia-gtc-ai-jensen-huang-cpu-gpu.html)，理解 Vera CPU + Rubin GPU 的协同架构

## 🔧 Skill 开发专家
