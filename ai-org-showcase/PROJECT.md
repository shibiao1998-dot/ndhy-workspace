# PROJECT.md — ai-org-showcase

## 概述
AI组织分享会的交互式演示网页。一个单文件 HTML 页面，展示"一个人 + 七个 AI = 完整产品团队"的组织化 AI 工作方式。

## 技术栈
- 纯 HTML + CSS + Vanilla JS（单文件，无外部依赖）
- Canvas 粒子动画（hero 背景）
- IntersectionObserver（滚动触发动画）
- CSS glassmorphism（玻璃拟态卡片效果）
- 响应式设计（移动端适配）

## 关键决策
- [2026-03-14] 单文件架构，所有 CSS/JS 内联 — 原因：离线可用，双击即开
- [2026-03-14] 粒子动画用 IntersectionObserver 控制启停 — 原因：离开 hero 区时停止动画，节省性能
- [2026-03-14] 对比章节采用手风琴交互 — 原因：6个维度全展开太长，手风琴聚焦阅读体验
- [2026-03-14] 角色卡片同时支持 hover（桌面）和 click（移动）展开 — 原因：兼顾两端交互习惯
