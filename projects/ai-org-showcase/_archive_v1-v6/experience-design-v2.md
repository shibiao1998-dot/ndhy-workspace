# 体验设计方案 — ai-org-showcase v2

## 一、信息架构

### 页面结构（单页长滚动）
单页应用，通过滚动驱动叙事。不做多页切换，减少跳出。

```
[Hero 首屏] — 震撼开场 + 核心数据
    ↓ 滚动
[组织全貌] — 26 专家阵列，分领域展示
    ↓ 滚动
[核心创新] — 五大颠覆性创新
    ↓ 滚动
[对比维度] — AI 组织 vs 传统方式
    ↓ 滚动
[活证据] — 这个网站本身的制作过程
    ↓ 滚动
[CTA 尾部] — 行动召唤
```

### 导航
- 顶部固定导航栏（滚动后出现），包含各板块锚点
- Logo / 品牌标识在左，导航链接在右
- 移动端：汉堡菜单

## 二、交互设计方向

### 设计基调
**深色 + 科技感 + 粒子/线条动效**——对标 Stripe、Linear、Vercel 等世界级产品官网。

### 核心交互模式

1. **Hero 首屏**
   - 全屏深色背景 + 粒子网格动画（暗示神经网络/连接）
   - 主标题大字动态出现（打字机效果或渐入）
   - 核心数据点数字滚动动画（1 → 26 → 6 → 0）
   - 向下滚动提示箭头

2. **组织全貌**
   - 专家卡片网格布局
   - 滚动进入视口时逐个动画出现（staggered fade-in）
   - 卡片 hover 展示详细信息
   - 按领域分组，组标题 + 组卡片
   - 可选：流水线连线图（展示协作关系）

3. **核心创新**
   - 每个创新一个独立区块
   - 滚动触发内容动画（从左/右/下方滑入）
   - 流水线创新：动画展示流程节点依次点亮
   - 权力分层：分层卡片堆叠动画
   - 记忆体系：层级示意图
   - 自愈机制：三级恢复流程动画
   - 知识治理：三层架构示意

4. **对比维度**
   - 左右对比布局（传统 vs AI 组织）
   - 滚动触发逐行出现
   - 视觉对比：灰暗（传统） vs 明亮（AI 组织）

5. **活证据**
   - 时间轴/流程图展示网站制作过程
   - 每个参与专家的角色卡片

6. **CTA**
   - 简洁有力，渐变背景
   - 明确行动按钮

### 响应式策略
- **桌面**（>1024px）：完整动效，多列布局
- **平板**（768-1024px）：简化动效，两列/单列
- **手机**（<768px）：最简动效，单列，卡片全宽，触控友好

### 性能策略
- 动画使用 CSS transform + opacity（GPU 加速）
- 粒子效果使用 Canvas，控制粒子数量
- 滚动动画使用 Intersection Observer（非 scroll 事件）
- 图片懒加载（如果有的话）
- 首屏内容 SSG 或内联，< 3 秒加载

## 三、状态处理

### 页面状态
| 状态 | 处理 |
|------|------|
| Loading | 简洁加载动画（品牌 logo 或骨架屏） |
| Success | 正常内容展示 |
| Slow Network | 先展示文字内容，动效降级 |
| JS Disabled | 纯 HTML/CSS 静态版可读 |

### 交互反馈
- 导航点击：平滑滚动到目标区域
- 卡片 hover：微微上浮 + 光影变化
- CTA 按钮：hover 发光效果

## 四、体验语言

### 风格
- **自信但不浮夸**——用结构和数据证明，不用形容词堆砌
- **简洁有力**——短句，信息密度高
- **技术感**——适度使用技术语言，不过度通俗也不过度专业

### 文案规范
- 标题：陈述事实，不用感叹号
- 描述：一句话说清楚核心，不超过两行
- 按钮：动词优先（"看看怎么运转" 而非 "了解更多"）
- 数字：突出展示，自带说服力

## 五、前端交接说明

### 技术建议
- 推荐 Next.js（SSG）或 Astro（静态站生成器），或纯 HTML/CSS/JS
- 技术选型由前端开发专家最终决定
- 动效库建议：Framer Motion / GSAP / 原生 CSS 动画
- 粒子效果：tsParticles 或自定义 Canvas

### 页面清单
| 页面/区块 | 核心目标 |
|-----------|---------|
| Hero | 30 秒内传达核心信息，制造震撼 |
| 组织全貌 | 展示 26 专家的规模和专业分工 |
| 核心创新 | 深入理解 5 大颠覆性创新 |
| 对比维度 | 清晰对比优势 |
| 活证据 | 用网站本身证明实力 |
| CTA | 转化行动 |

### 关键约束
1. 首屏加载 < 3 秒
2. 移动端体验不打折——核心内容完整可读
3. 深色主题为主（#0a-#1a 色系）
4. 动效节制——服务于信息传达，不是炫技
5. 无障碍基础：对比度达标、键盘可导航
