## 状态定义

| 状态 | 含义 |
|------|------|
| 🟢 活跃 | 正在推进中 |
| 🟡 暂停 | 暂时搁置，等待条件或决策 |
| 🔵 已完成 | 已交付验收 |
| ⚫ 归档 | 已归档，不再维护 |

## 新建项目规范

1. **命名**：英文小写-连字符，如 `my-new-project`
2. **必须包含**：`PROJECT.md`（参考 `PROJECT-TEMPLATE.md` 模板）
3. **创建后**：在本文件的项目列表中新增一行
