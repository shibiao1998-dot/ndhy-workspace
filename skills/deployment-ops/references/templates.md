# 发布交付包模板

## 概览

发布交付包包含 7 个标准文档：

```
{project}/deployment/
├── deploy-plan.md                  # 发布方案
├── release-checklist.md            # 发布检查清单
├── env-config-checklist.md         # 环境配置检查清单
├── rollback-plan.md                # 回滚方案
├── observability-checklist.md      # 可观测性检查清单
├── post-release-watch-plan.md      # 发布后观察计划
└── release-risk-summary.md         # 发布风险总结
```

## 2. release-checklist.md 模板

```markdown
# 发布检查清单

## 基本信息
- **项目名称**：
- **发布版本**：v{x.y.z}
- **检查日期**：{YYYY-MM-DD}
- **检查人**：

## 发布前检查

### 构建与产物
- [ ] 前端构建产物版本正确，hash校验通过
- [ ] 后端镜像/包版本正确
- [ ] 数据库迁移脚本已准备（如适用）
- [ ] 构建产物已推送到目标仓库/镜像库

### 上游结论
- [ ] 联调结论：通过（附链接/编号）
- [ ] 测试结论：可发布（附链接/编号）
- [ ] 代码审查：无阻断项（附链接/编号）
- [ ] 已知缺陷已评估，可接受范围明确

### 环境与配置
- [ ] 环境一致性已确认（开发→预发→生产）
- [ ] 环境变量已配置且值正确
- [ ] 密钥/证书有效且未过期
- [ ] 部署账号权限已确认
- [ ] 网络策略/防火墙规则已配置
- [ ] 数据库连接/迁移权限已确认

### 依赖服务
- [ ] 第三方API可达性已验证
- [ ] 数据库/缓存/消息队列连通性已验证
- [ ] 依赖服务版本兼容性已确认

### 发布保障
- [ ] 发布步骤已文档化
- [ ] 回滚方案已制定
- [ ] 健康检查已配置
- [ ] 监控告警已配置
- [ ] 发布观察计划已制定

## 发布后检查
- [ ] 健康检查全部通过
- [ ] 核心链路验证通过
- [ ] 监控指标正常
- [ ] 无异常告警
- [ ] 日志无异常错误

## 发布结论
- **结论**：✅ 发布成功 | ⚠️ 带风险运行 | ❌ 已回滚
- **备注**：
```
