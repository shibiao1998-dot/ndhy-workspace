# 发布策略 — 发布检查清单

### 产品准备
- [ ] 功能开发全部完成，无遗留项
- [ ] 所有P0/P1级bug已修复
- [ ] 已知P2/P3级bug已评估影响
- [ ] 产品验收通过
- [ ] 预发布环境完成最终验证

### 技术准备
- [ ] 数据库变更已准备并验证
- [ ] 回滚方案已确认可执行
- [ ] 灰度配置已就绪
- [ ] 监控告警已部署
- [ ] 性能压测通过
- [ ] 安全检查通过

### 用户准备
- [ ] 培训内容已准备（视频/文档/指南）
- [ ] 帮助中心已更新
- [ ] FAQ已准备并发给客服团队
- [ ] 用户通知已准备（文案已审核）

### 运营准备
- [ ] 客服团队已培训
- [ ] 客服排班已调整（发布后首周加强）
- [ ] 反馈收集渠道已准备
- [ ] 应急联系人名单已确认

### 沟通准备
- [ ] 内部发布通知已准备
- [ ] 外部发布公告已准备
- [ ] 更新日志已撰写
- [ ] 重要客户1对1通知已安排

### 应急准备
- [ ] 回退方案已确认，参与人已知晓
- [ ] 应急响应团队已就位
- [ ] 紧急联系人电话清单已发放
- [ ] 回退触发条件已明确定义
- [ ] 如需停服，维护页面已准备
