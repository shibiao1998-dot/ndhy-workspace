## 旗舰文本模型

### Claude Opus 4.6 (`claude-opus-4-6`)
- **定位**：编程+Agentic 最强旗舰
- **核心 Benchmark**：
  - SWE-Bench Verified: **81.4%**（全球 #1）
  - Chatbot Arena ELO: **1503**（全球 #1 用户偏好）
- **强项**：
  - 复杂多文件软件工程（Claude Code 主力）
  - 长上下文保真度极高
  - 创意写作、多轮对话、风格适应
  - 安全性和对齐最好
  - Agentic 多步骤任务解决
  - Context Compaction（自适应推理+上下文压缩）
- **弱项**：成本高、速度不如 GPT-5.4
- **最佳用途**：ACP 开发任务、代码审查、架构设计、需求分析

### GPT-5.4 (`gpt-5.4-2026-03-05`)
- **定位**：最强通用主义者
- **核心 Benchmark**：
  - 赢 5/12 个基准测试类别（vs Claude 赢 3 个）
  - 但 Claude 在编程/推理/代码质量核心维度领先
- **强项**：
  - 成本效率好、速度快
  - Computer Use 原生支持
  - 高吞吐管线
  - 知识工作广度
- **弱项**：复杂多文件编程略逊 Claude
- **最佳用途**：通用对话、文档处理、高吞吐批量任务

### Claude Sonnet 4.6 (`claude-sonnet-4-6`)
- **定位**：Tier 1.5 高性价比
- **强项**：长文本处理、速度快、成本适中
- **最佳用途**：长文档审查、批量任务、技术文档、测试专家

### Gemini 3.1 Pro Preview (`gemini-3.1-pro-preview`)
- **定位**：超长上下文之王
- **强项**：1M+ token 上下文窗口、多模态
- **最佳用途**：超长文档分析、大型代码库理解

## 推理专精模型

### O3 (`o3-2025-04-16`)
- **定位**：推理天花板
- **核心 Benchmark**：ARC-AGI 准确率 3x O1
- **强项**：数学、科学、结构化多步骤分析
- **成本**：高（扩展推理时间）
- **最佳用途**：复杂数学证明、架构决策论证、科学分析

### O4-Mini (`o4-mini-2025-04-16`)
- **定位**：推理高性价比之王
- **核心 Benchmark**：最高实时编码分数
- **强项**：数学+编码组合、200K 上下文、视觉支持
- **成本**：比 O3 便宜得多
- **最佳用途**：STEM 批量任务、高性价比推理

### DeepSeek R1 (`DeepSeek-R1-0528`)
- **定位**：开源推理最强
- **核心 Benchmark**：MMLU 90.8%、GPQA Diamond 71.5%
- **强项**：显示完整推理链、可审计、成本极低
- **最佳用途**：需要透明推理过程的分析任务

## 中文强势模型

### Doubao Seed 2.0 Pro (`doubao-seed-2-0-pro-260215`)
- **定位**：中文旗舰（国内第一）
- **核心 Benchmark**：
  - SuperCLUE 2025: **69.16**（国内第一、全球第二）
  - AIME25: **98.3**（数学推理全球顶级）
  - LiveCodeBench: **87.8**
  - Codeforces: **3020**
  - 多模态能力全球前三
- **强项**：中文理解#1、数学推理极强、多模态
- **弱项**：代码生成和指令遵循有提升空间
- **最佳用途**：中文内容创作、数学题解、中文对话

### Qwen 3.5 Plus (`qwen3.5-plus-2026-02-15`)
- **定位**：极速中文高手
- **核心 Benchmark**：
  - IFBench: **76.5**（指令遵循顶级）
  - GPQA Diamond: **88.4**
  - 响应速度: Claude Sonnet 1/6
- **强项**：极快速度、指令遵循、中文好
- **最佳用途**：高吞吐中文批量任务、快速文本处理

### Qwen 3 Max (`qwen3-max-2025-09-23`)
- **强项**：中文综合能力强
- **最佳用途**：中等复杂度中文任务

### Kimi K2 (`kimi-k2-250905`)
- **定位**：Agent 能力突出
- **架构**：1T 总参数 MoE、32B 激活
- **强项**：SWE-bench 编码好、前端开发好、Agent/Swarm 模式
- **最佳用途**：前端开发辅助、Agent 任务
