# 视觉层级与留白规则

> 用户不会阅读页面——他们扫描。视觉层级决定用户先看什么。

## 阅读模式

### F 型阅读模式

```
████████████████████████
████████████████████
██████████████
████████████████████████
████████████████
███████████
```

**特点**：用户先水平扫描顶部，然后目光下移，扫描范围逐渐变窄。
**适用**：文字密集型页面（文章、文档、搜索结果）。
**设计要点**：
- 重要内容放在顶部和左侧
- 每段的前两个词最关键
- 用小标题打断长文本，创造新的水平扫描起点

### Z 型阅读模式

```
█████████────────█████████
         ╲
          ╲
           ╲
█████████────────█████████
```

**特点**：用户目光沿 Z 形路径移动——左上→右上→左下→右下。
**适用**：内容少、视觉驱动型页面（Landing Page、营销页）。
**设计要点**：
- 左上角放 Logo/品牌
- 右上角放导航/CTA
- 中间放核心信息
- 右下角放主要行动按钮

## 留白层级

### 三级留白体系

| 级别 | 用途 | 推荐间距 | 视觉效果 |
|------|------|---------|---------|
| **L1 区块间留白** | 页面大区块之间 | 48-96px | 强分隔，明确换区 |
| **L2 模块间留白** | 同一区块内模块之间 | 24-32px | 中分隔，内容分组 |
| **L3 元素间留白** | 模块内元素之间 | 8-16px | 弱分隔，内容关联 |

### 留白比例规则

| 规则 | 说明 |
|------|------|
| **关联越紧，间距越小** | 标题与段落 = 8px，段落与段落 = 16px，板块与板块 = 48px |
| **层级越高，留白越大** | H1 上方 > H2 上方 > H3 上方 > 段落上方 |
| **外间距 > 内间距** | 卡片之间的间距 > 卡片内部的间距（否则看不出卡片边界） |
| **不对称留白** | 元素上方留白 > 下方留白（标题距上方内容更远，距下方内容更近） |

### 信息密度档位

| 档位 | 基础间距 | 适用场景 | 每屏信息量 |
|------|---------|---------|-----------|
| **紧凑型** | 4-8px | 数据表格、管理后台 | 高 |
| **舒适型** | 8-16px | 标准产品页面 | 中 |
| **宽松型** | 16-32px | 营销页、展示页 | 低 |

## 视觉重心引导

### 引导手段

| 手段 | 原理 | 示例 |
|------|------|------|
| **大小对比** | 大元素先被注意 | 数字指标用 36px，标签用 14px |
| **颜色对比** | 高对比色先被注意 | 主按钮用 primary 色，次按钮用 ghost |
| **留白包围** | 周围留白多的元素更突出 | CTA 按钮上下留足空间 |
| **位置优先** | 左上 > 右上 > 左下 > 右下 | Logo 在左上，CTA 在右上 |
| **运动/动画** | 动态元素吸引注意 | 新消息用轻微抖动提示 |
| **分组聚合** | 相关内容视觉聚合 | 用卡片框住一组相关信息 |

### 一屏一焦点原则

```
✅ 正确：每个视口高度内有一个主焦点
┌─────────────────────────┐
│                         │
│    【核心指标】 ← 焦点    │
│                         │
│   辅助说明              │
│                         │
└─────────────────────────┘

❌ 错误：同一视口内多个焦点竞争注意力
┌─────────────────────────┐
│  【指标A】  【指标B】     │
│  【图表C】  【警告D】     │
│  【通知E】  【按钮F】     │
└─────────────────────────┘
```

## 视觉层级清单

| 元素 | 推荐样式 | 说明 |
|------|---------|------|
| 页面标题 (H1) | 30-36px / bold / fg-primary | 每页只有一个 |
| 区块标题 (H2) | 24px / semibold / fg-primary | 区分大板块 |
| 模块标题 (H3) | 18-20px / semibold / fg-primary | 模块内分组 |
| 正文 | 16px / normal / fg-primary | 主要阅读内容 |
| 辅助文字 | 14px / normal / fg-secondary | 说明、提示 |
| 标签/元数据 | 12px / normal / fg-tertiary | 时间戳、标签 |
| 主按钮 | primary 背景 + inverse 文字 | 页面核心操作 |
| 次按钮 | outline / ghost 样式 | 辅助操作 |
