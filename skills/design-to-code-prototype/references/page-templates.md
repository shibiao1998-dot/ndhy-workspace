# 常见页面类型原型模板

> 不从零开始——选择最接近的模板，基于它定制。

## 模板索引

| 模板 | 适用场景 | 核心区块 |
|------|---------|---------|
| Dashboard | 数据概览、运营后台首页 | 统计卡片 + 图表 + 数据表 |
| 列表页 | CRUD 管理、资源浏览 | 筛选栏 + 数据表/卡片 + 分页 |
| 表单页 | 创建/编辑数据 | 表单分组 + 校验 + 提交 |
| 详情页 | 查看单条数据详情 | 面包屑 + 基本信息 + 关联数据 |
| Landing Page | 产品介绍、营销着陆 | 英雄区 + 特性展示 + CTA |

## 1. Dashboard 模板

### 页面结构

```
┌──────────────────────────────────────────────────┐
│  [Logo]  Dashboard          [搜索]  [通知] [头像] │ ← Header
├────────┬─────────────────────────────────────────┤
│        │  欢迎回来，张老师           2024-03-15    │ ← Page Header
│ 首页   ├─────────┬─────────┬─────────┬───────────┤
│ 课程   │ 总学生   │ 活跃课程 │ 完成率   │ 本周新增  │ ← Stats
│ 实验   │ 156     │ 4       │ 87.5%  │ +12      │
│ 学生   ├─────────┴─────────┴─────────┴───────────┤
│ 分析   │                                         │
│ 设置   │  [== 趋势图表 ==]   [排行榜]              │ ← Charts
│        │                                         │
│        ├─────────────────────────────────────────┤
│        │  最近活动                        [查看全部]│
│        │  ┌──────┬────────┬──────┬───────┐      │ ← Table
│        │  │ 名称  │ 状态    │ 日期  │ 操作   │      │
│        │  └──────┴────────┴──────┴───────┘      │
└────────┴─────────────────────────────────────────┘
```

### 响应式变换
- **Desktop**：Sidebar 展开 + Stats 4列 + Chart 8:4 双栏
- **Tablet**：Sidebar 折叠为图标 + Stats 2列 + Chart 单列
- **Mobile**：无 Sidebar（汉堡菜单）+ Stats 1列 + Chart 单列 + Table 转卡片

## 2. 列表页模板

### 页面结构

```
┌──────────────────────────────────────────────────┐
│  [Logo]  Navbar                    [通知] [头像]  │
├────────┬─────────────────────────────────────────┤
│        │  课程管理              [+ 新建课程]       │ ← Page Header
│ Sidebar├─────────────────────────────────────────┤
│        │  [搜索...]  [状态▾]  [科目▾]  [重置]     │ ← Filter Bar
│        ├─────────────────────────────────────────┤
│        │  □ 课程名称    科目   状态    学生数  操作 │ ← Table Header
│        │  □ 力学基础    物理   进行中  42     ···  │
│        │  □ 光学实验    物理   已完成  38     ···  │
│        │  □ 化学反应    化学   草稿    0      ···  │
│        │  □ 生物细胞    生物   进行中  51     ···  │
│        ├─────────────────────────────────────────┤
│        │  共 24 条  ◀ 1 2 3 ... 6 ▶   每页 10 ▾  │ ← Pagination

```

### 响应式变换
- **Desktop**：完整表格展示
- **Tablet**：隐藏次要列（科目），保留核心列
- **Mobile**：Table 转卡片列表，每条数据一张卡片

## 模板选择指南

| 你要做的页面 | 选择模板 | 主要定制 |
|-------------|---------|---------|
| 后台首页/概览 | Dashboard | 修改统计指标和图表类型 |
| 数据管理（CRUD） | 列表页 | 修改表头和筛选条件 |
| 创建/编辑表单 | 表单页 | 修改表单字段和分组 |
| 查看单条记录 | 详情页 | 修改信息区块和关联数据 |
| 产品官网/营销 | Landing Page | 修改文案和特性展示 |
| 登录/注册 | 表单页（简化版） | 居中单卡片布局 |
| 设置/个人中心 | 详情页（Tab 切换） | 左侧导航 + 右侧表单 |
