# 学习心理学

> 聚焦：心理学原理在教育产品功能设计中的直接应用。

## 1. 认知负荷理论（Sweller）

**三种认知负荷**：

| 类型 | 定义 | 产品设计对策 |
|------|------|-------------|
| 内在负荷 | 学习材料本身的复杂度 | 元素交互性分析 → 合理分解/排序 |
| 外在负荷 | 不当的呈现方式造成的额外负担 | 减少冗余信息、优化信息呈现 |
| 相关负荷 | 促进深层加工的有益认知活动 | 增加练习、自解释、类比活动 |

**关键设计原则**：

| 原则 | 说明 | 产品实现 |
|------|------|---------|
| 分割原则 | 复杂信息分步呈现 | 分步教程、渐进式内容展开 |
| 冗余原则 | 避免同时呈现重复信息 | 动画 + 旁白时不再叠加完整文字 |
| 空间邻近原则 | 相关信息物理位置靠近 | 标注与图示紧邻，不分离 |
| 时间邻近原则 | 相关信息同时呈现 | 解说与动画同步，不先后分离 |
| 信号原则 | 用视觉线索引导注意力 | 高亮、箭头、颜色编码 |
| 预训练原则 | 先学关键概念再学整体 | 核心术语预学习模块 |
| 模态原则 | 图 + 旁白 > 图 + 文字 | 实验演示用语音解说而非字幕 |

**常见违反（教育产品中的高频问题）**：
- ❌ 一个页面同时放实验操作、数据表格、理论解释和习题
- ❌ 3D 实验界面元素过多，操作按钮和信息标签密集堆叠
- ✅ 分阶段展示：先观察 → 再操作 → 后分析

## 2. 自我决定理论（Deci & Ryan）

**三种基本心理需求**：

| 需求 | 定义 | 产品设计实现 |
|------|------|-------------|
| 自主性 | 感觉行为是自己选择的 | 提供学习路径选择、允许跳过已掌握内容 |
| 胜任感 | 感觉自己能做到 | 适当难度挑战、及时正向反馈、进步可视化 |
| 归属感 | 感觉与他人有联系 | 小组协作、同伴互动、社区归属 |

**动机连续体（产品应推动的方向）**：

```
无动机 → 外部调节 → 内摄调节 → 认同调节 → 整合调节 → 内在动机
  ❌      外部奖惩     内疚/焦虑    认为重要     与自我整合    纯粹乐趣
         (积分排名)    (怕落后)    (知道有用)    (这就是我)   (探索本身)
```

**产品设计警告**：
- ⚠️ 过度使用外部奖励（积分、排行榜）会**削弱内在动机**（过度理由效应）
- ✅ 奖励应指向**能力反馈**（"你掌握了 X"）而非**行为控制**（"做了 Y 得 Z 积分"）

## 3. 心流理论（Csikszentmihalyi）

**心流条件**：
1. 明确的目标
2. 即时的反馈
3. 挑战与技能的匹配（略高于当前水平）

**产品设计中的心流通道**：

```
        高 ┌──────────┐
挑       │  焦虑区   │
战       │    ↗ 心流  │
难       │   ↗  通道  │
度       │  ↗       │
        │ 无聊区    │
        低 └──────────┘
            低 ——→ 高
             技能水平
```

**设计要点**：
- 难度自适应：持续保持在心流通道内
- 即时反馈：操作后 < 1 秒给出反应
- 清晰目标：每个学习单元有明确的、可达成的小目标
- 减少干扰：学习过程中减少无关弹窗和通知

## 4. 间隔重复与遗忘曲线（Ebbinghaus）

**艾宾浩斯遗忘曲线关键数据**：
- 20分钟后遗忘 42%
- 1小时后遗忘 56%
- 1天后遗忘 74%
- 1周后遗忘 77%
- 1月后遗忘 79%

**间隔重复算法设计原则**：

| 原则 | 说明 |
|------|------|
| 间隔递增 | 每次正确回忆后，下次复习间隔拉长 |
| 难度自适应 | 错误/犹豫的内容缩短间隔，熟练的拉长 |
| 交错练习 | 混合不同主题的复习优于单一主题连续练习 |
| 提取练习 | 主动回忆 > 被动复习（测试效应） |
| 间隔基准 | 首次学习后：1天 → 3天 → 7天 → 14天 → 30天（可根据表现调整） |

**证据强度**：✅ 强。间隔重复是认知心理学中证据最充分的学习策略之一。

## 5. 测试效应（Retrieval Practice）

**核心发现**：主动提取记忆（测试）的学习效果远优于被动复习（重读）。

**产品设计启示**：
- 学习后立即安排**低压测试**（不计分/不排名的自测）
- 把"练习题"定位为**学习工具**而非**考核工具**
- 提供**自由回忆**练习（开放题）比**再认**练习（选择题）效果更好

## 6. 具身认知（Embodied Cognition）

**核心观点**：认知不只发生在大脑中，身体动作参与认知加工。

**产品设计启示（尤其 3D/VR 产品）**：
- 让学生**动手操作**虚拟实验器材而非观看演示
- 手势交互、拖拽操作比点击按钮更有利于空间概念学习
- VR 中的**身体移动**（走近观察分子结构）比平面缩放更有利于理解

**证据强度**：⚠️ 中等。具身认知的理论框架被广泛接受，但具体产品设计效果的实证研究仍在积累中。

## 7. 元认知（Metacognition）

**两个核心成分**：
- **元认知知识**：知道自己知道什么、不知道什么
- **元认知调控**：计划、监控、评估自己的学习过程

**产品设计实现**：

| 元认知成分 | 产品功能 |
|-----------|---------|
| 自我评估 | 学前自评（"你觉得自己掌握了多少？"） |
| 计划 | 学习计划制定工具、目标设定 |
| 监控 | 学习进度仪表盘、知识掌握热力图 |
| 评估 | 学后反思提示（"哪里还不清楚？"）、错题分析 |

**设计要点**：元认知能力本身需要培养——低年级学生（小学）元认知能力弱，需要更多系统引导。
