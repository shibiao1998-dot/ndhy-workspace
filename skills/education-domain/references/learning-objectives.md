# 学习目标分类与设计

> 聚焦：如何设计可评估、可达成的学习目标，指导教育产品的内容规划和评估体系。

## 1. 布鲁姆修订版分类学（Anderson & Krathwohl, 2001）

### 认知过程维度（六级）

| 层级 | 认知过程 | 关键动词 | 产品中的体现 |
|------|---------|---------|-------------|
| 1. 记忆 | 识别、回忆 | 列举、识别、说出 | 选择题、填空题、术语匹配 |
| 2. 理解 | 解释、举例、分类、推断 | 解释、举例、比较 | 概念解释题、分类任务、图表解读 |
| 3. 应用 | 执行、实施 | 计算、操作、使用 | 计算题、实验操作、方法运用 |
| 4. 分析 | 区分、组织、归因 | 区分、分析、对比 | 数据分析、因果推理、实验设计 |
| 5. 评价 | 检查、批判 | 判断、评价、论证 | 实验方案评估、论证分析 |
| 6. 创造 | 产生、计划、建构 | 设计、创造、提出 | 开放性实验设计、方案创造 |

### 知识维度（四类）

| 类型 | 说明 | 示例 |
|------|------|------|
| 事实性知识 | 基本元素和术语 | 化学元素符号、物理量单位 |
| 概念性知识 | 分类、原理、理论 | 质量守恒定律、牛顿三定律 |
| 程序性知识 | 方法、步骤、技术 | 实验操作步骤、计算方法 |
| 元认知知识 | 关于认知的知识 | 学习策略、自我评估能力 |

### 二维分类表的使用

目标应同时指定**认知过程**和**知识类型**：

| | 事实性 | 概念性 | 程序性 | 元认知 |
|---|---|---|---|---|
| 记忆 | 记住化学元素 | | | |
| 理解 | | 解释质量守恒 | | |
| 应用 | | | 操作滴定实验 | |
| 分析 | | 分析反应类型 | | |
| 评价 | | | | 评价自己的实验方案 |
| 创造 | | | 设计新实验方案 | |

**产品设计启示**：
- 学习目标应覆盖**多个层级**，不只停留在记忆和理解
- 目标层级决定了**评估方式**——高层级目标不能用选择题评估
- 不同学段的目标层级分布应有差异（低学段偏低层级，高学段偏高层级）

## 2. 新课标目标体系

### 目标层次结构

```
课程目标（核心素养）
  └── 学段目标（1-2年级、3-4年级、5-6年级、7-9年级）
        └── 内容要求（具体知识和技能）
              └── 学业要求（质量标准）
```

### 课标目标的动词体系

**新课标使用分层动词表达不同的学业要求**：

| 层次 | 知识类目标 | 技能类目标 | 情感类目标 |
|------|----------|----------|----------|
| 了解/感受 | 说出、识别、描述 | 观察、尝试 | 感受、体验 |
| 理解/认同 | 解释、说明、比较 | 操作、测量 | 认同、欣赏 |
| 应用/内化 | 分析、论证、设计 | 设计、评价 | 践行、坚持 |

### 产品内容对标课标的方法

1. **知识点映射**：产品内每个知识点标注对应的**课标内容要求**编号
2. **目标层级标注**：每个学习活动标注对应的**认知层级**
3. **学业质量对标**：评估标准参照**课标学业质量标准**描述
4. **素养指标关联**：标注每个活动培养哪些**核心素养**

## 3. SOLO 分类理论（Biggs & Collis）

**五个理解水平**：

| 水平 | 描述 | 学生表现 | 评估中的体现 |
|------|------|---------|-------------|
| 前结构 | 未理解 | 答非所问、胡猜 | 完全不相关的回答 |
| 单一结构 | 理解一个方面 | 只说出一个要素 | "温度升高反应变快" |
| 多元结构 | 理解多个方面但未关联 | 列举多个要素 | "温度、浓度、催化剂都影响反应速率" |
| 关联结构 | 理解要素间的关联 | 整合解释 | "温度升高→粒子动能增大→有效碰撞增多→反应速率增大" |
| 拓展抽象 | 能迁移和创造 | 超越当前情境 | "基于这个原理，如果要设计一个…" |

**产品设计启示**：
- SOLO 可用于**开放题的自动/半自动评分**——判断学生回答处于哪个水平
- 学习目标应定义**期望达到的 SOLO 水平**
- 教学活动应设计为引导学生从**低水平向高水平跃迁**

## 4. 学习目标编写规范

### ABCD 法

| 要素 | 说明 | 示例 |
|------|------|------|
| A - Audience（学习者） | 谁在学 | 初中三年级学生 |
| B - Behavior（行为） | 可观察的行为 | 能独立设计实验方案 |
| C - Condition（条件） | 在什么条件下 | 给定实验器材和已知条件 |
| D - Degree（标准） | 达到什么程度 | 方案包含假设、变量控制和数据记录方法 |

### 常见目标编写错误

| 错误类型 | 错误示例 | 修正示例 |
|----------|---------|---------|
| 用教师行为代替学生行为 | "讲解质量守恒定律" | "学生能用实验证据解释质量守恒定律" |
| 目标不可观察 | "理解牛顿第三定律" | "能举例说明牛顿第三定律的三种应用情境" |
| 目标过于宽泛 | "掌握化学实验技能" | "能独立完成溶液配制操作，浓度误差 ≤ 5%" |
| 缺少评估标准 | "学会数据分析" | "能从实验数据中画出正确的图表并标注趋势线" |

## 5. 目标与评估的对齐

**一致性原则**：目标层级必须与评估方式匹配。

| 目标层级 | 适合的评估方式 | 不适合的评估方式 |
|----------|-------------|----------------|
| 记忆/理解 | 选择题、填空题、简答题 | — |
| 应用 | 计算题、操作任务 | 纯概念选择题 |
| 分析/评价 | 案例分析、实验报告、论述题 | 选择题（难以评估深层思维） |
| 创造 | 开放性项目、实验设计 | 任何封闭式题型 |

**产品设计关键**：如果产品声称培养"高阶思维能力"，但评估只有选择题——**目标与评估不一致**。
