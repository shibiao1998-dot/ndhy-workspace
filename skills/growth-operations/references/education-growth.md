# 教育行业增长特别指南

## 一、学期周期——教育增长的时间节奏

### 核心认知
> 以"学期"为单位思考，不是以"月"为单位。

### 教育增长年历

| 时段 | 增长重点 | 核心动作 | 关键指标 |
|------|---------|---------|---------|
| **学期初**（前 2 周） | 获取 + 激活 | 新学期准备活动、新用户引导、校级部署 | 新增学校数、激活率 |
| **学期中**（3-14 周） | 留存 + 活跃 | 习惯养成、教学赋能活动、内容运营 | WAU、功能使用深度 |
| **考试前**（15-16 周） | 活跃峰值 | 复习工具专题、资源推送 | 使用频次、活跃度 |
| **学期末**（17-18 周） | 数据沉淀 + 续费 | 学期总结报告、续费决策影响 | 续费意向、学期留存 |
| **寒/暑假** | 培训 + 预热 | 教师成长活动、下学期预热 | 培训参与率、预约率 |

### 学期关键节点运营策略

**学期初——黄金获取窗口**
```
→ 教师在制定教学计划，最容易接受新工具
→ 新学期 = 新开始，使用意愿最高
→ 动作：
  - 新学期准备活动（帮教师快速完成课程设置）
  - 新用户集中培训
  - 老用户学期衔接（零成本过渡到新学期）
```

**学期中——深化使用的关键期**
```
→ 日常教学进行中，产品需要融入工作流
→ 习惯是在这个阶段养成的
→ 动作：
  - 持续的教学赋能活动
  - 功能使用引导和技巧推送
  - 教研分享活动
```

**学期末——续费决策的关键期**
```
→ 学校评估本学期的工具使用效果
→ 决定下学期是否续费/扩展
→ 动作：
  - 学期使用报告（数据化展示价值）
  - 续费沟通（提前 1 个月启动）
  - 收集改进建议
```

**假期——教师发展窗口**
```
→ 教师相对空闲，适合培训和学习
→ 为下学期做准备
→ 动作：
  - 教师成长系列活动
  - 新功能预热和培训
  - 下学期准备工具包
```

## 二、决策使用分离——教育增长的核心挑战

### "鸡和蛋"问题

```
教师用产品需要有好的内容/资源
       ↑                    ↓
好的内容/资源需要有教师贡献
```

### 冷启动策略
1. **平台先投入**：做一批高质量的种子内容/模板
2. **运营活动激励**：通过活动引导教师贡献内容
3. **逐步建立 UGC 生态**：从 PGC → PUGC → UGC

### 决策者 vs 使用者

| 维度 | 决策者（教育局/校长） | 使用者（教师） |
|------|---------------------|---------------|
| 关心什么 | 效果、数据、合规 | 好不好用、省不省时间 |
| 说服方式 | 数据报表、政策对齐、案例背书 | 实际使用体验、同伴推荐 |
| 增长策略 | 一条线搞定（让他们愿意买/推） | 一条线搞定（让他们愿意用/持续用） |

### 两条线并行策略
```
决策者线：案例报告 → 数据展示 → 政策对齐 → 购买/推广
                                                ↕ 相互影响
使用者线：试用体验 → 啊哈时刻 → 习惯养成 → 口碑传播
```

**最好的情况**：使用者用得好 → 反过来影响决策者续费/扩展。

## 三、口碑传播——教育行业的增长引擎

### 教育行业口碑传播的特点

| 特点 | 说明 | 对增长的影响 |
|------|------|-------------|
| **信任度高** | 同行推荐 >> 广告 | 口碑获取的用户质量最高 |
| **传播慢但稳** | 不会一夜爆发，但持续积累 | 需要耐心，但复利效应强 |
| **传播路径明确** | 校内 → 校际 → 区域 | 可以有针对性地推动 |
| **负面传播快** | 一次失误可能影响一片区域 | 保护口碑 = 保护增长根基 |

### 口碑传播的催化剂

1. **可量化的效果**："用了这个产品，备课时间减少 40%"
2. **可讲述的故事**：真实教师的使用故事，有情感共鸣
3. **可展示的成果**：学生作品、教学案例、数据报告
4. **可参与的场景**：教研活动、分享会、公开课

### 口碑传播路径设计
```
单个教师用得好
     ↓ 校内分享
同教研组教师开始用
     ↓ 教研活动
同校更多教师使用
     ↓ 校长圈子
校长向其他学校推荐
     ↓ 区域交流
区域教育局关注 → 区域推广
```

## 四、种子教师——增长飞轮的启动器

### 种子教师的价值
> 一个真心认可产品的种子教师，价值大于 100 个被动注册的用户。

### 种子教师的特征

| 特征 | 表现 |
|------|------|
| 内在动力 | 主动探索新功能，不需要反复提醒 |
| 教学热情 | 用产品是为了提升教学，不是为了完成任务 |
| 影响力 | 在学校/区域有话语权，同行愿意听他的 |
| 反馈意愿 | 愿意提出真实反馈，帮助产品改进 |

### 种子教师培育路径
```
发现 → 建立连接 → 深度支持 → 产出成果 → 扩大影响
```

| 阶段 | 动作 | 产出 |
|------|------|------|
| **发现** | 从活跃数据中筛选、从培训中发现 | 种子教师候选名单 |
| **建立连接** | 一对一沟通、专属群组 | 种子教师社群 |
| **深度支持** | 优先体验新功能、专属培训 | 深度使用能力 |
| **产出成果** | 引导产出教学案例、使用经验 | 可传播的内容 |
| **扩大影响** | 分享会、公开课、案例展示 | 影响更多教师 |

### 种子教师管理原则
1. **不要太多**：每所学校 2-3 个足够，重质量不重数量
2. **不要强迫**：种子教师是自愿的，强迫会适得其反
3. **给荣誉不给钱**：教育行业"被当做工具人"很反感
4. **持续关注**：定期沟通，了解需求变化
5. **允许退出**：种子教师不是终身制，允许自然流动

## 五、教育增长的 5 个特殊约束

| # | 约束 | 含义 | 增长策略影响 |
|---|------|------|-------------|
| 1 | **学期周期** | 一切以学期为节奏 | 运营日历必须和教学日历对齐 |
| 2 | **决策使用分离** | 买的人 ≠ 用的人 | 需要两条增长线并行 |
| 3 | **教师时间稀缺** | 教师没有多余时间 | 任何运营动作都要问"占用多少时间？" |
| 4 | **口碑为王** | 同行推荐是第一获取渠道 | 保护口碑 > 短期增长 |
| 5 | **阶梯式增长** | 不是指数级，以区域为台阶 | 要有耐心，每个区域要做透 |

## 六、教育增长的时间预期

### 合理的增长预期

| 阶段 | 时间 | 合理预期 | 不合理预期 |
|------|------|---------|-----------|
| 探索期 | 1-2 学期 | 找到 PMF，10-20 所种子学校 | "3个月百万DAU" |
| 成长期 | 2-4 学期 | 100+ 活跃学校，WAU 稳定增长 | "指数级增长" |
| 规模期 | 2+ 学年 | 区域覆盖，自然增长开始显现 | "全国一夜铺开" |

### 教育增长的正确节奏
```
阶梯式增长是正常的：
→ 每攻下一个区域/学校群，上一个台阶
→ 台阶之间可能有平台期，这是正常的
→ 平台期是巩固和优化的时间
→ 急功近利的增长手段在教育行业通常会反噬
```
