## 实验设计模板

```markdown
# 增长实验 #[编号]：[实验名称]

## 假设
"如果我们做了[X改变]，目标用户的[Y行为]会提升[Z%]，因为[原因]。"

## 实验设计
→ 变量：（一次只改一个）
→ 实验组：______
→ 对照组：______
→ 样本量：______（需满足统计显著性）
→ 实验周期：______（至少一个完整教学周）

## 目标指标
→ 主指标：______
→ 辅助指标：______
→ 护栏指标：______（确保没有负面影响）

## 执行计划
| 阶段 | 时间 | 动作 | 负责人 |
|------|------|------|--------|
| 准备 | | | |
| 执行 | | | |
| 分析 | | | |

## 判断标准
→ 成功：主指标提升 ≥ [X]%，且护栏指标无下降
→ 失败：主指标无显著变化或下降
→ 待定：主指标有提升但未达阈值 → 延长实验或扩大样本

## 结果记录
→ 实际结果：______
→ 分析：______
→ 决策：推广 / 迭代 / 放弃
→ 学到了什么：______
```

```markdown
| 时段 | 运营重点 | 关键动作 |
```
