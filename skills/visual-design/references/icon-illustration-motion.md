# 视觉设计 — 图标、插画与动效

## 图标风格定义

### 风格选择

| 风格 | 特征 | 适用场景 |
|------|------|---------|
| **线性(Outline)** | 1.5~2px 描边，无填充 | 工具类产品、轻量感 |
| **面性(Filled)** | 纯色填充 | 活跃状态、强调 |
| **双色(Duotone)** | 前景+背景两层色 | 品牌感强、装饰性 |

**铁律：同一产品内只用一种主风格。** 面性图标可作为线性图标的活跃态变体，但不混用双色与线性。

### 图标网格规范

| 参数 | 值 | 说明 |
|------|-----|------|
| **基础网格** | 24×24px | 标准图标画布 |
| **活跃区域** | 20×20px | 内容区域（留 2px 安全边距） |
| **笔触粗细** | 1.5px 或 2px | 统一不混用 |
| **端点** | Round Cap | 圆角端点 |
| **拐角** | Round Join | 圆角拐角 |
| **最小细节** | 2px | 小于 2px 的细节在小尺寸下不可辨认 |

### 图标尺寸变体

| Token | 尺寸 | 用途 |
|-------|------|------|
| **icon-xs** | 12px | 角标、状态指示 |
| **icon-sm** | 16px | 按钮/输入框内图标 |
| **icon-md** | 20px | 导航项图标、列表图标 |
| **icon-lg** | 24px | 独立功能图标 |
| **icon-xl** | 32px | 空状态、特征图标 |
| **icon-2xl** | 48px | 品牌展示、大特征 |

### 图标颜色规则

| 规则 | 说明 |
|------|------|
| **默认色** | neutral-600（与辅助文字同级） |
| **交互色** | 跟随文字颜色变化（hover/active） |
| **活跃色** | brand-primary-500（当前选中态） |
| **语义色** | 对应语义色（成功/警告/错误/信息） |
| **禁止** | 不给图标加独立于文字的颜色 |

## 插画风格定义

### 风格维度

| 维度 | 选项 | 说明 |
|------|------|------|
| **风格** | 扁平 / 2.5D / 手绘 / 几何抽象 | 选择一种，全产品统一 |
| **人物比例** | 写实(1:7) / 萌系(1:5) / 无人物 | 与品牌调性匹配 |
| **线条** | 有描边 / 无描边 | 配合图标风格 |
| **色彩** | 仅使用品牌色板 | 不引入色板外颜色 |

### 插画使用场景

| 场景 | 尺寸 | 说明 |
|------|------|------|
| **空状态** | 120~200px | 列表/搜索/收藏为空时，给情感温度 |
| **新手引导** | 200~300px | 引导页、功能介绍 |
| **错误页面** | 200~300px | 404/500/断网 |
| **品牌展示** | 灵活 | 登录页、关于页 |
| **成功反馈** | 80~120px | 操作完成确认 |

### 插画质量标准

1. 色彩来源于品牌色板（允许在色板基础上调透明度）
2. 风格与图标风格协调（同产品视觉语言）
3. 有意义（传达信息或情感，非纯装饰）
4. 不遮挡文字内容
5. 提供 SVG 格式，支持暗色模式适配

## 动效视觉语言

### 动效目的分类

| 类别 | 目的 | 示例 |
|------|------|------|
| **反馈** | 告诉用户"操作被接收了" | 按钮按下缩放、checkbox 勾选 |
| **过渡** | 帮助用户理解"发生了什么变化" | 页面切换、面板展开 |
| **引导** | 引导用户注意力 | 新通知弹出、高亮闪烁 |
| **品牌** | 传达品牌个性 | 启动动画、Logo 动画（极少使用） |

### 缓动函数

| Token | 函数 | 用途 |
|-------|------|------|
| **ease-default** | cubic-bezier(0.25, 0.1, 0.25, 1) | 通用默认 |
| **ease-in** | cubic-bezier(0.4, 0, 1, 1) | 元素退出（加速离开） |
| **ease-out** | cubic-bezier(0, 0, 0.2, 1) | 元素进入（减速到达） |
| **ease-in-out** | cubic-bezier(0.4, 0, 0.2, 1) | 元素移动（位置变化） |
| **ease-bounce** | cubic-bezier(0.34, 1.56, 0.64, 1) | 弹性效果（极少使用） |

### 时长阶梯

| Token | 值 | 用途 |
|-------|-----|------|
| **duration-instant** | 50ms | 色彩/透明度微变 |
| **duration-fast** | 100ms | 按钮状态切换、开关切换 |
| **duration-normal** | 200ms | 下拉展开、Tab 切换 |
| **duration-slow** | 300ms | 面板展开、模态框进入 |
| **duration-slower** | 400ms | 页面过渡 |
| **duration-slowest** | 500ms | 复杂场景过渡（极少使用） |

### 过渡类型

| 类型 | 适用场景 | 参数 |
|------|---------|------|
| **淡入淡出(Fade)** | 内容切换、Toast 出现 | opacity 0↔1, ease-out, normal |
| **滑入滑出(Slide)** | 面板、抽屉、菜单 | translateX/Y, ease-out, slow |
| **缩放(Scale)** | 弹窗出现、按钮按下 | scale 0.95↔1, ease-out, normal |
| **展开折叠(Expand)** | 手风琴、折叠面板 | height/max-height, ease-in-out, slow |
| **组合** | 弹窗 = Fade + Scale | 同时执行 |

### 动效铁律

1. **功能 > 装饰**：每个动效必须有明确功能目的
2. **时长 ≤ 300ms**：超过 300ms 用户会感到等待（极特殊情况例外）
3. **可关闭**：尊重 prefers-reduced-motion 系统设置
4. **不阻塞**：动效不阻挡用户操作
5. **一致性**：同类交互使用同样的动效参数
6. **节制**：一个页面同时播放的动效不超过 2 个

## 动效 Token 产出格式

```
// Easing
ease-default: cubic-bezier(0.25, 0.1, 0.25, 1)
ease-in: cubic-bezier(0.4, 0, 1, 1)
ease-out: cubic-bezier(0, 0, 0.2, 1)
ease-in-out: cubic-bezier(0.4, 0, 0.2, 1)

// Duration
duration-instant: 50ms
duration-fast: 100ms
duration-normal: 200ms
duration-slow: 300ms
duration-slower: 400ms

// Transition Presets
transition-fade: opacity {duration-normal} {ease-out}
transition-slide: transform {duration-slow} {ease-out}
transition-scale: transform {duration-normal} {ease-out}
transition-expand: max-height {duration-slow} {ease-in-out}
```
