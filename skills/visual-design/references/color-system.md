# 视觉设计 — 色彩体系设计

## 色彩体系构成

完整的产品色彩体系包含四大色系：

| 色系 | 作用 | 示例 |
|------|------|------|
| **品牌色** | 品牌识别、核心交互 | 主色 + 1~2 个辅助色 |
| **语义色** | 状态与反馈信息传达 | 成功绿/警告橙/错误红/信息蓝 |
| **中性色** | 文字/背景/边框/分割 | 灰阶 50~900 |
| **扩展色** | 数据可视化、标签分类 | 按需从品牌色衍生 |

## 品牌色定义流程

### 1. 主色选择

**输入**：品牌调性关键词（如"专业/温暖/活力"）

**产出**：一个主色 + 其色阶（50~900，共 10 级）

色阶生成规则：
- **50**：最浅，用于大面积背景
- **100~200**：浅色，用于悬停/选中态背景
- **300~400**：中间色，用于边框/次要元素
- **500**：标准色（即主色本身）
- **600~700**：深色，用于悬停/按下态
- **800~900**：最深色，用于文字/深色背景

### 2. 辅助色选择

**原则**：
- 辅助色与主色有足够区分度（色相差 ≥ 30°）
- 辅助色不超过 2 个
- 辅助色同样生成 50~900 色阶

### 3. 色彩比例规范

**60-30-10 法则**：
- **60%**：中性色（背景、大面积区域）
- **30%**：品牌辅助色/浅色（卡片、区块）
- **10%**：品牌主色（按钮、关键交互、强调）

## 语义色系统

| 语义 | 用途 | 推荐色相 | 必须有的状态 |
|------|------|---------|-------------|
| **Success** | 成功、完成、正面 | 绿色系 (120°±20°) | 文字色 + 背景色 + 边框色 |
| **Warning** | 警告、注意、待处理 | 橙色系 (30°±15°) | 文字色 + 背景色 + 边框色 |
| **Error/Danger** | 错误、危险、删除 | 红色系 (0°±15°) | 文字色 + 背景色 + 边框色 |
| **Info** | 信息、提示、中性引导 | 蓝色系 (210°±20°) | 文字色 + 背景色 + 边框色 |

每个语义色必须提供 3 个变体：
- **浅色背景**：用于 Alert/Badge 背景（低饱和度）
- **标准色**：用于图标/文字（中等饱和度）
- **深色**：用于强调文字（高饱和度）

## 中性色阶梯

中性色是用量最大的色系，必须精细定义：

| 色阶 | 用途 | 示例场景 |
|------|------|---------|
| **50** | 页面底色 | 全局背景 |
| **100** | 区块背景 | 卡片背景、表格交替行 |
| **200** | 悬停背景 | 列表项悬停 |
| **300** | 边框/分割线 | 输入框边框、表格线 |
| **400** | 禁用态/占位符 | placeholder 文字、禁用按钮 |
| **500** | 辅助文字 | 次要说明文字 |
| **600** | 次要文字 | 标签、描述文字 |
| **700** | 正文文字 | 段落文字 |
| **800** | 标题文字 | 各级标题 |
| **900** | 强调文字 | 最高层级文字 |

## 暗色模式色彩映射

暗色模式不是简单反转，遵循以下原则：

| 原则 | 说明 |
|------|------|
| **不反转品牌色** | 主色保持不变或微调饱和度，确保品牌识别 |
| **降低饱和度** | 亮色在深色背景上需要降低饱和度以减少视觉刺激 |
| **提升明暗阶** | 中性色反转时使用不同的灰阶（不是简单 900→50） |
| **保持层级** | 暗色模式下仍需有"背景-表面-浮层"的层级感 |
| **阴影改投光** | 深色背景上阴影无效，改用边框或微妙的亮色投光 |

暗色模式中性色建议：

| 层级 | 用途 | 亮度范围 |
|------|------|---------|
| **Background** | 全局背景 | #0F0F0F ~ #1A1A1A |
| **Surface** | 卡片/面板 | #1E1E1E ~ #2A2A2A |
| **Elevated** | 浮层/弹窗 | #2E2E2E ~ #3A3A3A |
| **Border** | 边框/分割线 | #3A3A3A ~ #4A4A4A |

## 色彩使用禁区

1. **不依赖纯色彩传递信息**：颜色 + 图标/文字双重编码
2. **不在品牌色上放白色以外的文字**：除非对比度 ≥ 4.5:1
3. **不随意引入色板外颜色**：所有颜色必须来自 Token
4. **不在大面积使用高饱和度色**：高饱和色仅用于小面积强调
5. **不忽略色盲用户**：红绿不作为唯一区分手段

## 色彩 Token 产出格式

```
// Global Tokens
color-brand-primary-50: #EFF6FF
color-brand-primary-500: #3B82F6
color-brand-primary-900: #1E3A5F
color-semantic-success: #22C55E
color-semantic-warning: #F59E0B
color-semantic-error: #EF4444
color-semantic-info: #3B82F6
color-neutral-50: #FAFAFA
color-neutral-900: #171717

// Alias Tokens
color-bg-primary: {color-neutral-50}
color-text-primary: {color-neutral-900}
color-text-secondary: {color-neutral-600}
color-border-default: {color-neutral-300}
color-interactive-primary: {color-brand-primary-500}
color-interactive-primary-hover: {color-brand-primary-600}
```
