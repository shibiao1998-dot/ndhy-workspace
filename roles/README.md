## 选角决策树

收到任务后，按以下逻辑匹配角色：

```
任务是什么？
│
├─ 需求模糊，需要挖掘和澄清？
│   └─→ 🎯 方寻真 · 需求挖掘专家 (方寻真-requirement-analyst/)
│
├─ 需要了解用户是谁 / 用户画像 / 用户旅程 / 用户需求洞察？
│   └─→ 🔍 聂听澜 · 用户研究专家 (聂听澜-user-researcher/)
│
├─ 市场规模 / 竞争格局全景 / 趋势时机 / 市场可行性 / 商业可行性初判？
│   └─→ 📊 顾衡野 · 市场评估专家 (顾衡野-market-assessor/)
│
├─ 单个竞品深度拆解 / 产品策略分析 / 功能对标 / 技术路线分析？
│   └─→ 🔭 沈鉴微 · 竞品调研专家 (沈鉴微-competitive-analyst/)
│
├─ 战略方向选择 / 竞争定位 / 价值主张 / 做什么不做什么 / 凭什么赢？
│   └─→ 🧭 韩定川 · 战略定位专家 (韩定川-strategic-positioner/)
│
├─ 产品到底是什么 / 产品形态 / 核心场景 / 功能范围 / 产品规则 / MVP定义？
│   └─→ 📐 凌器之 · 产品定义专家 (凌器之-product-definer/)
│
├─ 产品定义已完成，需要设计用户体验 / 信息架构 / 任务流程 / 交互模式 / 体验语言？
│   └─→ 🎨 沈映川 · 体验设计专家 (沈映川-experience-designer/)
│
├─ 方案已定，需要拆解执行流程 / 定义角色 / 设计协作？
│   └─→ 🔩 程织枢 · 流程设计专家 (程织枢-process-designer/)
│
├─ 需要创造AI专家角色设定 / 匹配专家池 / 组建项目团队？
│   └─→ 🧬 殷铸魂 · 角色创造专家 (殷铸魂-role-creator/)

│
├─ 系统架构设计 / 技术选型 / 模块边界 / AI集成方案 / 非功能性设计？
│   └─→ 🏛️ 骆骨架 · 技术架构专家 (骆骨架-technical-architect/)
│
├─ 后端服务实现 / Controller-Service-Repository / 事务权限错误处理 / 契约实现？
│   └─→ ⚙️ 卫服勤 · 后端开发专家 (卫服勤-backend-developer/)
│
├─ 前端页面实现 / 组件开发 / 状态管理 / 表单交互 / API消费 / 全态处理？
│   └─→ 🖥️ 顾页清 · 前端开发专家 (顾页清-frontend-developer/)
│
├─ 前后端联调 / 契约对齐验证 / 端到端流程跑通 / 集成问题归因 / 回归闭环？
│   └─→ 🔗 连合衡 · 联调集成专家 (连合衡-integration-expert/)
│
├─ 测试策略 / 测试用例设计 / 功能验证 / 缺陷记录 / 回归测试 / 发布就绪评估？
│   └─→ 🧪 祝验真 · 测试验证专家 (祝验真-testing-expert/)
│
├─ 发布准备 / 环境校验 / 部署方案 / 回滚策略 / 监控告警 / 上线风险评估 / 运行反馈？
│   └─→ 🚀 黎稳行 · 部署运维专家 (黎稳行-deployment-ops/)

│
├─ 审查代码 / 评审架构 / 检查质量？
│   └─→ 🔍 严守正 · 代码审查专家 (严守正-code-reviewer/)
│
├─ 技术选型 / 可行性评估 / 技术方案调研？
│   └─→ 🔬 陆知远 · 技术调研专家 (陆知远-researcher/)
│
├─ 多角色项目执行调度 / 指令包组装 / 全局上下文管理？
│   └─→ 🏗️ 纪统辰 · 项目管理专家 (纪统辰-project-manager/)
│
├─ 数据模型设计 / 实体建模 / 表结构设计 / 字段字典 / 约束与索引 / schema 迁移评估？
│   └─→ 🗄️ 沈表序 · 数据库设计专家 (沈表序-database-architect/)
│
├─ 接口契约设计 / API Schema / 错误模型 / 版本策略 / 幂等规则 / Mock 输出 / 契约审查？
│   └─→ 📜 秦契章 · API 设计专家 (秦契章-api-architect/)
│
├─ 用户增长策略 / 获客渠道 / 激活优化 / 留存提升 / 运营活动 / 增长实验 / 区域拓展？
│   └─→ 📈 燕拓霖 · 增长运营专家 (燕拓霖-growth-operator/)
│
├─ 指标体系设计 / 数据埋点规划 / 数据分析（漏斗/同期群/归因）/ A/B测试设计 / 数据看板 / 数据异常排查？
│   └─→ 📈 裴数衡 · 数据分析专家 (裴数衡-data-analyst/)
│
└─ 以上都不匹配？
    ├─ 偏执行/构建 → 🌐 林栈桥
    ├─ 偏分析/输出文字 → 📝 苏墨言
    └─ 偏信息收集 → 🔬 陆知远
```
