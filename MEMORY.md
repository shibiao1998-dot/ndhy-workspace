- **审查任务必须给够时间 + 要求先做自动化扫描**。审查超时=审查未完成=漏检。超时设置 ≥ 开发任务的50%；task中明确要求先跑全局扫描（编码\uFFFD、格式、链接），再做人工审查。（2026-03-14 乱码漏检教训）

- **每次生成内容后必须做乱码检查**。中转站接入 API 的固有问题，无法从源头修复，只能在产出后扫描修复。这是强制性流程，不是可选项。（2026-03-14 老板明确要求）
